# Island Sokoban
Sokoban box-moving game. [Play it here!](https://www.kleemans.ch/island-sokoban)

## Install

To install dependencies (Phaser 3 etc.) run

    npm install

## Run locally

For running the game locally, you can use

    npm run dev

which will start the game. Simple as that!
